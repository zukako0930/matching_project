# matching_projectとは
お試しでマッチングアプリのようなものを作ってみたいと思い作成に至る。
<br>マッチングアプリのロジックを習得しておくと今後のWebアプリ制作に役立つ事間違いなし。

# 目標
いわゆる人対人のマッチングアプリで必要となる基本的な機能を実装する。
- メッセージ
- マッチのリクエスト
- プロフィール管理(画像、プロフィール文)

# 発展
せっかくPythonおよびRail上での外部コマンド実行の知見があるので、機械学習によるレコメンド機能も追加したい。
